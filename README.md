# 📝 Task Manager Backend API

A simple REST API for managing tasks using Node.js, Express, and PostgreSQL.

---

## 🚀 Features

- Create a task
- Get all tasks
- Update a task
- Mark task as completed
- Delete a task
- Centralized error handling
- Input validation middleware

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- PostgreSQL
- pg (node-postgres)

---
